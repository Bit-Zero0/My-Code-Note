# C++中 constexpr关键字的作用是什么

在C++11中，`constexpr`关键字的功能是使指定的常量表达式获得在程序编译阶段计算出结果的能力，而不必等到程序运行阶段。它可用于修饰普通变量、函数（包括模板函数）以及类的构造函数。
